STM32 environment on Visual Studio Code

Originally published on GitHub Gists This guide will help you install and setup Visual Studio Code for programming and debugging STM32 boards. I tested this guide under Arch Linux and Ubuntu 18.04. If you get it to work under other setups please let me know so I will update the steps with more info. Warning If you were using STM32CubeIDE or SystemWorkbench before, you need to convert your projects in order for them to work.
