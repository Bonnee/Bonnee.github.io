Using GitHub Actions is like programming in the '60s

If you were lucky enough to have access to a computer in the 1960s you would have had to go through a convoluted process in order to run programs on it. At first you would have had to punch your program onto cards, which can store 80 characters each (yes, the 80 char line width of ttys and programming came from punch cards). Then, while making sure to keep the cards in order, you would have had to go to the computer room and give the card stack to the computer operator, who would have put them in an execution queue and eventually would have loaded them into the machine to run your program.
